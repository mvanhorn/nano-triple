# Nano Triple

3 images, one prompt, instant A/B/C comparison. Pick the winner or refine any version.

## Install

```
clawhub install nano-triple
```

Or copy `SKILL.md` to `~/.openclaw/skills/nano-triple/SKILL.md`

## Features

- **3 parallel variations** - Same prompt, three distinct takes from Nano Banana Pro's natural randomness
- **Style presets** - Photorealistic, illustration, watercolor, minimalist, gradient, retro, neon, sketch, anime, pixel-art
- **Iterative refinement** - "I like B but make it more colorful" generates 3 new variations from your pick
- **Image editing** - Edit existing images with natural language
- **Aspect ratios** - 16:9, 1:1, 9:16, 4:3 presets for any platform

## Requirements

- `GEMINI_API_KEY` - Get one at [Google AI Studio](https://aistudio.google.com)
